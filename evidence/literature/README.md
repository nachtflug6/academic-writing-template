# Literature

Literature notes in this folder must remain traceable to the original source.

## Minimum standard

- Every literature note should identify the source clearly enough to recover it.
- Each claim, observation, or excerpt should be linked to a source.
- Include a page number or other precise locator whenever available.
- If wording is taken directly from the source, include the literal text and mark it clearly as a quotation or excerpt.
- Keep direct quotation separate from your interpretation or summary.
- If a locator is missing, mark that explicitly with a TODO rather than leaving the support ambiguous.

## Preferred structure

For source-close extracts, prefer entries like:

- Source: author, year, title, and any local identifier available
- Page: page number or equivalent locator
- Excerpt: exact source wording in quotation marks
- Why it helps: brief note on what the excerpt supports

For summaries, make sure each synthesized point can still be traced back to one or more sources, ideally with page references for the strongest supporting passages.

## Example

Use any source note in `extracts/` as a model when it links claims to specific sources, includes page-level identification, and preserves literal source text for key supporting passages.
