## Submission Checklist

- [ ] All chapters drafted and included
- [ ] Formatting meets all requirements (margins, fonts, spacing)
- [ ] Page numbering correct and consistent
- [ ] Table of contents matches actual content
- [ ] All figures/tables numbered and captioned
- [ ] All figures/tables referenced in text
- [ ] Bibliography complete and properly formatted
- [ ] All citations match bibliography entries
- [ ] No orphaned sections or missing files
- [ ] Cross-references all resolve correctly
- [ ] Spell check and grammar pass complete
- [ ] Compiles to PDF without errors
- [ ] PDF file named per requirements
- [ ] Title page and front matter correct
- [ ] Abstract within length limit
- [ ] Acknowledgments included (if required)
- [ ] Dedication included (if required)
- [ ] Glossary included (if required)
- [ ] Appendices included (if any)
- [ ] No confidential or personally identifying information (if blind review)
- [ ] Advisor approval obtained

- [ ] Final copy ready for binding (if required)
